Picture In Picture Mode

This is a course Project made in the course Artificial Intelligence (CS F407) at BITS Pilani. The broad idea for making this project comes under the domain of Intelligent Tutoring System wherein making it easy for the student to code side-by-side the instructor while watching any youtube coding video.
